# Images Directory

This directory is for storing website images.

## Directory Structure

- `logos/` - Location logos (Melbourne, Sydney, main brand logo)
- `hero/` - Hero section background images
- `food/` - Food photography for menu and content sections
- `icons/` - UI icons and graphics

## Image Guidelines

### Recommended Formats
- **Photos**: JPG or WebP
- **Logos**: SVG or PNG (with transparency)
- **Icons**: SVG preferred

### Optimization
- Compress images before uploading
- Use appropriate dimensions (don't upload oversized images)
- Consider using WebP format with JPG fallbacks for better performance

### Naming Convention
Use descriptive, lowercase names with hyphens:
- `hero-melbourne.jpg`
- `logo-main.svg`
- `food-steak-1.jpg`

## Adding Images to the Site

Update the relevant HTML files to reference your images:

```html
<!-- Example -->
<img src="images/hero/hero-melbourne.jpg" alt="Grill Americano Melbourne">
```

## Image Sources

To download images from the original website:
1. Visit https://grillamericano.com
2. Right-click on images and "Save Image As..."
3. Or use browser DevTools to find image URLs
4. Save to appropriate subdirectory

**Note**: Ensure you have proper rights/permissions to use any images you download.
